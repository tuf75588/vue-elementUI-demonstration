# Steps for complete application.

- [x] Use Vue-CLI to bootstrap application.
- [ ] Import Element-UI(for ui components) as well as axios using npm
- [ ] On the load of the Vue app, use axios to make a REST call to get a list of users.
- [ ] endpoint for users information [users-endpoint](https://jsonplaceholder.typicode.com/users)
- [ ] Use the Element-UI table to display the users
- [ ] Add 3 columns (Name, Username, Email Address)
- [ ] Provide functionality to Add, Remove and Edit users from this table.
